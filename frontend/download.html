Install Skill Coach VS Code Extension

Integrate AI-powered skill analysis directly into your development workflow.

Step 1 — Download Extension

Click below to download the VS Code extension package.

Download Extension (.vsix)

Step 2 — Install in VS Code

  1. Open VS Code
  2. Go to Extensions (Ctrl + Shift + X)
  3. Click three dots (…) → Install from VSIX
  4. Select downloaded file

Step 3 — Login Inside VS Code

  1. Open Command Palette (Ctrl + Shift + P)
  2. Run: Skill Coach: Login
  3. Enter your registered email & password
  4. Start coding 🎯
